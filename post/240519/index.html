前言

搭建一个独立的博客系统，分享自己的写作和技术，是一个很棒的事情。hugo基于go语言开发，是世界上最快的搭建博客的框架，上手也很简单。

hugo搭建博客的过程

  • 安装hugo 扩展版本,下载地址：hugo release
  • hugo new site mysite //创建站点的根目录
  • cd mysite
  • git init && git clone https://github.com/olOwOlo/hugo-theme-even themes/even //下载主题
  • cp ./themes/even/exampleSite/config.toml ./ mv config.toml hugo.toml //复制even下的配置文件
  • hugo new post/first_post.md //创建md文章
  • hugo server // 运行hugo
  • 通过http://localhost:1313访问博客

hugo修改配置和使用

到这里，基本的博客已经可以使用了，下面简单介绍一下hugo的一些使用和配置。

添加about页面

使用命令hugo new about.md创建about页面，打开about.md,添加如下修改

1
2
3
4
5
6
7

+++
title = 'About'
date = 2024-05-19T10:30:35+08:00
menu = 'main'
comment = false
weight = 60
+++

添加图片

使用hugo 命令hugo new post/newpost/index.md 创建newpost文件夹，在newpost文件夹下创建pics文件夹，把图片放到这个文件夹下。 在index.md中添加图片链接 ,即可显示图片，比如:

1

![desc](pics/1.jpg)

添加tags和categories

需要在config.toml添加下面的配置:

1
2
3
4

[taxonomies]
  tag = "tags"
  series = "series"
  category = "categories"

然后在md文档的开头添加tags和categories:

1
2

tags = ['hugo']
categories = ['hugo']
